# waydroid-opensuse-sources

This is source RPMs of Waydroid packages for openSUSE.
Maintained by jimed-rand.

# About
Since I have a initiation for maintaining the Waydroid packages and I need help from anyone to improve and fix the issues on the packages, I am publishing the sources for everyone to help me fix and improve. The sources will be upstreamed on my OBS repo for Waydroid, but feel free to use the sources for your own repo, but I suggest to made an contribution first to this repo as citation. So, help me fix the current issues for openSUSE users who want to run Waydroid by contribute to this repo!
